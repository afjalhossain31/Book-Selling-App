## ✨ Features

- 📖 Browse books by category  
- 🔍 Search and filter system  
- 🛒 Add to cart functionality  
- 💳 Simple payment system  
- 🔐 Firebase Authentication  
- ☁️ Real-time Database  
- 📱 Clean & responsive UI  

## 🛠️ Tech Stack

- Flutter  
- Dart  
- Firebase (Authentication & Database)
